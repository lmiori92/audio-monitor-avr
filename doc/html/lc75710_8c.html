void lc75710_write ( uint32_t  data)

This function writes the serial data (low-level) to the chip. The datasheet specifies about 0.5us delay between the edges of inputs. In any case, it has been tested that delays are not needed between calls, and this due to the particularly "slow" access to pins that the Arduino library operates. This note is left for a future bare-metal implementation.

Parameters
datathe 24-bit data to be sent over the serial line
